# Rick Roll Blocker

A browser extension that protects you from being Rick Rolled by blocking the famous Rick Astley video link.

## Features
- Blocks the specific Rick Roll URL
- Shows a fun "saved from April fool" message
- Tracks statistics of blocked attempts
- Toggle protection on/off
- Works across all websites

## Installation
1. Clone the repo with `git clone https://github.com/bandhan-majumder/Rick-Roll-Blocker`
2. Open Chrome/Edge browser
3. Go to chrome://extensions/ (or edge://extensions/)
4. Enable "Developer mode"
5. Click "Load unpacked"
6. Select the Rick-Roll-Blocker folder

## Files
- manifest.json - Extension configuration
- content.js - Main blocking logic
- blocked.html - Page shown when Rick Roll is blocked
- popup.html - Extension popup interface
- rules.json - URL blocking rules
